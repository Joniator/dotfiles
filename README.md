# Dotfiles

Managed with [chezmoi](https://chezmoi.io/).

# Installation

```sh
sh -c "$(curl -fsLS get.chezmoi.io)" && chezmoi init --apply Joniator --branch chezmoi
```

## Usage
Some programs need to be installed first:

### SDKMan
```sh
setup-sdkman
```

### Neovim
```sh
setup-nvim
```
